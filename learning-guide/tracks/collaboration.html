üéØ Exercise: Master 5 PM Prompt Patterns

Practice these essential prompting patterns that every PM should know:

1. Strategic Analysis Pattern
"Analyze [situation] from a product manager's perspective. Consider market context, user impact, technical feasibility, and business metrics. Provide 3 strategic recommendations with pros/cons for each."
2. User Story Generation Pattern
"Generate user stories for [feature]. Include persona details, acceptance criteria, and edge cases. Format as: As a [persona], I want [goal] so that [benefit]."
3. Competitive Analysis Pattern
"Compare [our approach] vs [competitor approaches] for [use case]. Include strengths, weaknesses, differentiation opportunities, and strategic recommendations."
